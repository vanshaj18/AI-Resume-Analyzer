What the Project Is

AI Resume Analyzer is a multi-signal evaluation system for candidate documents. It combines semantic understanding, technical depth checks, and psychometric heuristics to generate a concise summary and score for each resume or profile.

Results + Analytics

For async runs, the request is enqueued to Celery and executed as a three-step chain: extract text → analyze → report. Each stage updates progress for live UI tracking.

APIs

POST /analysis — single analysis (JSON payload).
POST /analysis/async — async pipeline (multipart PDF).
GET /analysis/status/{task_id} — progress updates.

Technical Services

Session Storage — UI results cached in Django sessions for fast refreshes.
Redis — stores PDFs, extracted text, and intermediate analysis results for async runs; also broker/backend for Celery.
Celery — background workers handle long-running pipelines; concurrency controlled via CELERY_CONCURRENCY.

LLM Providers

Groq — fast inference for short-form analysis and rapid feedback loops.
Gemini — strong reasoning and summarization; used for complex interpretation when enabled.

Analysis Modules

Semantic — role fit, relevance, and intent alignment across experience and keywords.
Technical — skills depth, stack maturity, project complexity, and implementation signals.
Psychometric — communication clarity, leadership signals, consistency, and polish.
